## 📝 Description

**DDOS** is a simple yet powerful UDP Flood DDoS testing tool written in Python. It allows users to perform stress tests on targets using UDP packet flooding.

> **⚠️ Educational & Testing Purposes Only**

---

## 🛠️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ipcscyber/erpl-ddos.git
   cd erpl-ddos
   ```

2. Make sure you have Python 3 installed.

3. (Optional) Install requirements if needed:
   ```bash
   pip install -r requirements.txt
   ```

---

## 🚀 Usage

```bash
python ddos.py
```

The script will prompt you to enter the target hostname or IP address.

### Example:
```
Masukkan hostname/Tn target: example.com
```
